# Jumpstart template repository

This is a template repository is used by the [Jumpstart Node.js project with a CI](https://github.com/AminaOti/Jump-start-Node.js-project-with-a-CI), to create a new Node.js project in the format of this template. To use this template, please see the instructions in [Jumpstart Node.js project with a CI](https://github.com/AminaOti/Jump-start-Node.js-project-with-a-CI)
